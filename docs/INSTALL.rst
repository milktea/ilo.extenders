ilo.extenders Installation
--------------------------

To install ilo.extenders using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ilo.extenders`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ilo.extenders

* Re-run buildout, e.g. with:

    $ ./bin/buildout
